# M7 Fitness Studio

A premium, responsive, dark-themed website for M7 Fitness Studio, located in Yousufguda, Kalyan Nagar, Hyderabad.

## Features
- **Modern Premium Design:** Inspired by industry-leading fitness centers, designed with dark layouts, vibrant accents, and clean typography.
- **Scroll Reveal Animations:** Interactive design with dynamic elements that animate smoothly upon scroll.
- **WhatsApp Chat Integration:** Direct, pulsing WhatsApp floating button for visitors to chat with the business owner.
- **Fully Responsive:** Optimised for desktop, tablet, and mobile screens.
- **SEO Ready:** Complete with meta tags, Open Graph tags, and structured JSON-LD schema for search engines.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)
- FontAwesome Icons

## Local Development
To preview the website locally, open `index.html` directly in any web browser, or launch a local HTTP server:
```bash
python -m http.server 8080
```
Then navigate to `http://localhost:8080`.
